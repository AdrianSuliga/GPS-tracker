# GPS-tracker
LTE-enhanced GPS tracker made on nRF91XX series.

Tracker is developed for both nRF9160DK and nRF9151DK. Unfortunately, I only have really old version of nRF9160DK - Enginnering A. For this reason
I need to bypass using nRF Modem Library when using it. That is why project is separated into ```nRF9160DK``` and ```nRF9151DK```.
